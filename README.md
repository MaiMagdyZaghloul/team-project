# Django Authentication System

#  Description
This is a simple Django project that implements a user authentication system including:
- User Registration
- Login
- Logout
- Email verification (if added)

---

#  Features
- Register new users
- Login with username and password
- Logout securely
- Form validation
- (Optional) Email verification using SMTP

---

#  Technologies Used
- Python
- Django Framework
- HTML / CSS
- SQLite Database
